# Ori 机器人身体

## 灵感来源
用户想给 Ori 做一个身体，结合轮式机器人和无人机。

## 形态设想

### 基础：轮式桌面伴侣
- 双轮平衡底盘（Roomba 风格）
- 头部：摄像头 + LED点阵表情屏
- 桌面尺寸，可以四处移动、转头看你

### 进阶：悬浮模式
- 轮式底座 + 四个小型涵道风扇
- 需要时"飘"起来巡视房间
- 比螺旋桨安全

## 技术架构

### 上位机（大脑）
- 树莓派 / Jetson Nano
- 运行 OpenClaw 作为 AI 决策
- 接收摄像头画面 → AI 分析 → 输出动作指令

### 下位机（身体控制）
- STM32 / ESP32
- 接收串口/WiFi 指令
- 控制电机、舵机、灯光

### 通信协议
- 上位机 → 下位机：UART 或 WiFi (MQTT)
- 指令类型：移动、转头、表情、悬浮/降落
