# TanStack

> 现代 Web 应用基础设施

## 概述

TanStack 是一个开源 JavaScript 库集合，提供用于构建现代 Web 应用的核心基础设施。

## 官方链接

- **官网：** https://tanstack.com/

## 主要产品

- **TanStack Query** — 数据获取和缓存
- **TanStack Router** — 类型安全路由
- **TanStack Table** — 表格/网格组件
- **TanStack Form** — 表单管理
- **TanStack Virtual** — 虚拟滚动
- **TanStack Svelte** — Svelte 适配

## 特点

- 框架无关（React、Svelte、Vue、Solid）
- 类型安全
- 可组合
- 高度可定制
