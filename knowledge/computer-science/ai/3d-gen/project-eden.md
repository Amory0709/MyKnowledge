# Project EDEN

> 3D 世界模型研究

## 概述

Project EDEN 是 Tripo AI 推出的世界模型研究计划，旨在构建持久的、可复用的、多人互动的 3D 环境。

## 官方链接

- **官网：** https://www.tripo3d.ai/research/project-eden

## 核心目标

- 持久化 3D 世界（persistent environments）
- 可复用的环境资产
- 多人互动（multiplayer interactive）
- 能根据用户和 Agent 的行为随时间演化

## 技术方向

- 从 2D 视频/图像到结构化 3D 状态的信号转换
- 结合游戏引擎提供精确的 3D 状态标注
- 支持动作响应、物理演化、场景逻辑学习
